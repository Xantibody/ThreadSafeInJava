# ThreadSafeInJava
Thread Safe In Java  
200000 Counter (100000 + 100000)

## Install


```bash
git clone https://github.com/Xantibody/ThreadSafeInJava.git
```

```bash
javac demo/Main.java
````

## Usage
```bash
# Thread Safe
java demo/Main.java SAFE
```
```bash
# Unsafe
java demo/Main.java ${other than "SAFE"}
```
